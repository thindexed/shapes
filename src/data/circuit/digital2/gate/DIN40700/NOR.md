# NOR Gate

## Description
A **NOR** gate is logically an inverted OR gate. 
It behaves according to the truth table on the bottom.


**Therefore, the output is always 0 except when all the 
inputs are 0.**

## Logic table

| INPUT 1   | INPUT   |  OUTPUT    |
|:---------:|:-------:|:----------:|
| Low       | Low     |  `High`       |
| `High`      | Low     |  Low       |
| Low       | `High`    |  Low       |
| `High`      | `High`    |  Low    |
